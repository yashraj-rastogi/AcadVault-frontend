# 🎓 AcadVault - Academic Achievement Management System

A modern web application for managing and showcasing academic achievements, built with Next.js 14 and TypeScript. **Now running with mock data - no backend required!**

## 🌟 Features

- **Achievement Management**: Submit, track, and manage academic achievements
- **Portfolio Generation**: Create professional portfolios from approved achievements  
- **Multi-Role Support**: Student, Faculty, and Admin dashboards
- **Mock Authentication**: Ready-to-use demo with sample data
- **Responsive Design**: Works seamlessly on desktop and mobile
- **Real-time Updates**: Interactive UI with instant feedback

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- npm or pnpm

### Installation & Run

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Start development server**
   ```bash
   npm run dev
   ```

3. **Open in browser**
   - Navigate to [http://localhost:3000](http://localhost:3000)
